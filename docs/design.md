# Bauhaus Design System

## Overview
The Bauhaus design system is characterized by simplicity, geometric shapes, and a limited color palette. It emphasizes function over form, using primary colors, bold typography, and clear grid-based layouts.

## Colors
- **Primary Red**: `#E32636`
- **Primary Blue**: `#00509E`
- **Primary Yellow**: `#FFD700`
- **Background (Light)**: `#F4F4F0`
- **Background (Dark)**: `#1A1A1A`
- **Text (Light Mode)**: `#1A1A1A`
- **Text (Dark Mode)**: `#F4F4F0`
- **Borders/Lines**: `#1A1A1A` (Light Mode), `#F4F4F0` (Dark Mode)

## Typography
- **Font Family**: Sans-serif (Inter, Helvetica, or Arial)
- **Headings**: Bold, all-caps or lowercase, large tracking (letter-spacing).
- **Body**: Regular weight, clean and legible.

## Shapes & Borders
- **Borders**: Thick, solid lines (e.g., `4px solid #1A1A1A`).
- **Border Radius**: Sharp corners (`0px`) or perfect circles (`50%`). No rounded rectangles.
- **Shadows**: Hard, solid shadows with no blur (e.g., `8px 8px 0px #1A1A1A`).

## Layout
- Grid-based, asymmetrical but balanced.
- Generous use of negative space (whitespace).
- High contrast between elements.

## Components
- **Buttons**: Rectangular, thick borders, solid background color (Red, Blue, or Yellow), hard shadow on hover.
- **Inputs**: Rectangular, thick borders, solid background color (White/Black), focus state uses a primary color border.
- **Cards**: Rectangular, thick borders, hard shadow, clear separation of content.
